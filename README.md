# SignalBrief Website

Official landing page for SignalBrief - AI-Powered Podcast Digests

## Deployment

This site is deployed via GitHub Pages at https://sigbrf.com

## Files

- `index.html` - Main landing page with pricing
- `success.html` - Post-checkout success page
- `update.html` - Subscription management page
- `CNAME` - Custom domain configuration

## Placeholders to Update

Before going live, replace these placeholders:

- `STRIPE_STARTER_LINK` - Stripe checkout URL for Starter plan
- `STRIPE_PRO_LINK` - Stripe checkout URL for Pro plan
- `STRIPE_PORTAL_LINK` - Stripe customer portal URL
- `INTAKE_FORM_LINK` - Google Form or intake URL
- `UPDATE_FORM_LINK` - Update podcast form URL
- `SUPPORT_EMAIL` - Support email address

## Local Development

Simply open `index.html` in a browser to preview locally.
